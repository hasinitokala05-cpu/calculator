# 🧮 Python Calculator

A simple command-line calculator built using Python. This project performs basic arithmetic operations through an interactive menu and is designed to help beginners understand Python functions, loops, conditionals, and user input.

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ❌ Division by zero handling
- 🔄 Menu-driven interface
- 🚪 Exit option

## 📂 Project Structure

```
Python-Calculator/
│
├── calculator.py
└── README.md
```

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your computer.

### Clone the Repository

```bash
git clone https://github.com/hasinitokala05-cpu/calculator.git
```

### Navigate to the Project

```bash
cd calculator
```

### Run the Program

```bash
python calculator.py
```
